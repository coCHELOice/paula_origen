# ORIGEN — Modern Editorial Build Spec

## Direction selected
Modern editorial

## Purpose
Build the ORIGEN landing page using the approved 9-section structure and the approved copy, with a premium editorial visual treatment.

## Fixed structure
1. Hero
2. Qué es ORIGEN
3. Para quién es
4. La experiencia
5. El lugar
6. Programa
7. Paula + equipo
8. Reserva
9. FAQ + cierre

## Fixed copy source
Use only:
project/02_copy/origen-landing-copy-v1.md

Do not rewrite or reinterpret the copy.

## Visual intent
A premium, sober, warm, contemporary editorial landing page.
The design should feel elevated and refined, but still easy to scan and clearly conversion-oriented.

## Hero layout
- asymmetrical balance
- large elegant typography
- generous whitespace
- editorial image adjacent to or slightly overlapping the text block
- immediate visibility of:
  - retreat name
  - location
  - date
  - primary CTA
  - secondary CTA

## Section rhythm
- generous vertical spacing
- subtle rhythm shifts across sections
- no heavy boxed segmentation
- off-white / pearl dominant background
- delicate divider lines where useful
- typography carries much of the hierarchy

## Visual system
- restrained palette
- off-white / pearl backgrounds
- charcoal or deep neutral text
- selective earthy accents only
- elegant high-contrast serif for headings
- clean geometric sans-serif for body copy and meta details

## UI / components
- borderless or near-borderless cards
- refined pricing presentation
- minimal FAQ accordion
- minimal list styling
- subtle 1px divider lines
- no visually loud shadows
- no bulky containers unless functionally necessary

## CTA treatment
- primary CTA: Reserva tu lugar
- secondary CTA: Ver programa
- minimalist premium buttons
- refined interaction states
- strong visibility without looking commercial or loud

## Mobile behavior
- preserve elegance on small screens
- maintain generous spacing
- clean image crops
- strong text hierarchy
- no clutter
- optional sticky “Reserva tu lugar” only if needed for usability

## Non-negotiables
- do not change the structure
- do not change the copy
- do not introduce new conceptual elements
- do not add extra sections
- do not reduce clarity in pursuit of aesthetics
- conversion must remain easy and visible
