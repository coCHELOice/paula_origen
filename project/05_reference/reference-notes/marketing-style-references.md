# Marketing Style References

## Miraval
Reference value:
- clear wellness structure
- strong booking logic
- clear inclusions
- useful section hierarchy
- balance between inspiration and functional clarity

Use for:
- landing structure
- benefits blocks
- booking / reserve logic
- inclusion clarity

Do not copy:
- corporate wellness tone
- overly hotel-like framing

## Aman
Reference value:
- sober luxury
- editorial restraint
- minimal premium atmosphere
- elegant spacing
- calm and exclusive visual rhythm

Use for:
- visual tone
- typography direction
- whitespace strategy
- premium restraint

Do not copy:
- extreme minimalism that hides practical conversion elements

## Application to ORIGEN
- structure discipline closer to Miraval
- visual restraint and premium tone closer to Aman
- keep ORIGEN’s feminine warmth, nature, and retreat identity
